HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • head -link rel="stylesheet" href="./assets/style.css" /> /head: this allows the .css to link to the html

CSS

  • The padding property is often confused with margin, but it is not the same thing. The padding property adds space around the content inside an element. When you increase padding, you are actually making the entire element bigger!

Git

  • git status: checks what branch you are on
  • git checkout -b feature/add-html: adds a new barnch with that specific name
  • git add -A: will commit/stage all files to the current branch
  • git commit -m "added code to HTML file" : commit to repository
  • git pull origin main: make sure brnach is up to date before opening a pull request
  • git push origin feature/add-html: push all of the changes
